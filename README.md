# CompilerWeb

编译原理课程实验的统一可视化前端。

## 当前内容

- **DFA 可视化**（实验一）：读入一个 DFA 定义（字母表 / 状态 / 起始态 / 接受态 / 转移矩阵），绘制状态转换图，并支持给定输入串后单步 / 自动模拟。
- **LR(0) 项目集规范族**（实验三）：输入文法后构造增广文法、闭包、Goto，绘制项目集与转移图，并标注接受态与冲突。

实验四（SLR(1) 分析表）在导航中占位，尚未实现。

## 运行方式

纯静态页面，无构建步骤。任选其一：

- 直接用浏览器打开 `index.html`。
- 或在目录下起一个静态服务器，例如：
  ```bash
  python -m http.server 8000
  ```
  然后访问 `http://localhost:8000/`。
