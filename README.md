# CompilerWeb

编译原理课程实验的统一可视化前端。

## 当前内容

- **DFA 可视化**（实验一）：读入一个 DFA 定义（字母表 / 状态 / 起始态 / 接受态 / 转移矩阵），绘制状态转换图，并支持给定输入串后单步 / 自动模拟。
- **LR(0) 项目集规范族**（实验三）：输入文法后构造增广文法、闭包、Goto，绘制项目集与转移图，并标注接受态与冲突。
- **SLR(1) 分析表**（实验四）：在 LR(0) 项目集规范族基础上计算 FIRST / FOLLOW 集，生成 ACTION / GOTO 分析表，并标注移进、规约、接受与冲突项。
- **语义分析可视化**（实验五）：基于课程语法分析源程序，展示 Token 流、抽象语法树、符号表与语义诊断，支持查看节点属性和错误位置。
- **中间代码生成可视化**（实验六）：复用实验五的语义分析结果，在无语义错误时生成四元式序列，展示临时变量、跳转回填结果和单条四元式说明。

各实验保持独立入口，实验五和实验六使用与课程 Lab 代码一致的固定语法。

## 运行方式

纯静态页面，无构建步骤。任选其一：

- 直接用浏览器打开 `index.html`。
- 或在目录下起一个静态服务器，例如：
  ```bash
  python -m http.server 8000
  ```
  然后访问 `http://localhost:8000/`。
